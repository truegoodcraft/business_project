## Key Features

*   **Inventory:** FIFO tracking, batch costs.
*   **Manufacturing:** Recipe-based runs, automatic cost calculation.
*   **Local Data:** SQLite database, you own your data.

## Getting Started

1.  Download & Install Python 3.10+.
2.  Clone Repo.
3.  Run `scripts/launch.ps1` (or `python launcher.py`).

## Dev Mode

Run `python launcher.py --dev` for console access.
